# team8.Bodhi-ki
Group Task Name: Student Team Members Management Application

1. Group Task Name: Student Team Members Management Application
2. Technologies and Software to be Used:
o Frontend (React.js)
o Backend (Node.js + Express + MongoDB)
o Visual Studio Code – Code editor for writing and managing frontend/backend code
o MongoDB Compass – GUI for managing MongoDB data
o Node.js – Backend JavaScript run time
3. App Structure – Pages &amp; Styling:
You have to create the following pages in your React application:
• Home Page
• Add Member Page
• View Members Page
• Member Details Page
You want to call (render) all these pages from your App.js using React Router. Also, apply basic CSS
styling so the pages have a consistent look.
4. Functional Requirements:
1. Display Team Name
o Show the team’s name on the homepage or header.
2. Add New Team Members
o Provide a form to input member details (e.g., name, role, contact info).
o Submit form data to the backend to store in MongoDB.
3. View All Team Members
o Fetch and display a list of all team members from the backend.
o Use cards or a table for a clean layout.
